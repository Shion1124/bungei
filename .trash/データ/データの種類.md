## オンチェーンデータ

- オンチェーンデータとは、ブロックチェーン上に直接記録される全ての情報です。仮想通貨の送金履歴、ウォレットのアドレスと残高、トランザクションの詳細（送信元、宛先、金額、手数料、タイムスタンプ）、スマートコントラクトの実行結果などが含まれます。

- ## ✅ 1. **Covalent**（https://www.covalenthq.com/）

	### 🔹 特徴
	
	- 全チェーン統一API（EVM系中心、SolanaやBaseなどにも対応）
	    
	- トークン保有、トランザクション履歴、DEX取引、NFT情報など取得可能
	    
	- REST APIベース、GraphQLではない
	    
	
	### 💰 無料枠（Free Tier）
	
	- **1,000 API calls/day**
	    
	- **最大5 calls/second**
	    
	- 一部チェーンは無料（例：Ethereum, Polygon, BNB Chain など）
	    
	- **Rate limit 超過時は429エラー**
	    
	
	### ✅ 無料枠で可能なこと
	
	- 特定アドレスのトークン残高やトランザクションの定期取得
	    
	- 小規模なDappや個人分析には十分
	    
	- API key 登録必須
	    
	
	### ❌ 無料枠の制限
	
	- 商用利用は不可（要Pro）
	    
	- 高頻度クエリには向かない（bot用途では注意）
	    
	- 高精度のDEXトレードデータは有料APIに含まれることも

- ## ✅ 2. **Moralis**（https://moralis.io/）

	### 🔹 特徴
	
	- Web3 APIに特化、特にNFT/ウォレット/イベント監視が得意
	    
	- REST API + webhookベースで通知も可能
	    
	- マルチチェーン（EVM系 + Solana）
	    
	
	### 💰 無料枠（Free Plan）
	
	- **5 million requests/month**
	    
	- **30 req/sec**
	    
	- **最大1万アドレスのトークン/NFT取得が無料**
	    
	- **Wallet API / Token API / NFT API** 全て無料利用可（一部制限付き）
	    
	
	### ✅ 無料枠で可能なこと
	
	- NFTの所有者一覧取得、メタデータ読み取り
	    
	- ウォレット単位での資産状況取得
	    
	- スマコンの呼び出し履歴取得
	    
	- リアルタイムでのトランザクション検知（Webhook）
	    
	
	### ❌ 無料枠の制限
	
	- 商用利用や大規模トラフィック用途は有償
	    
	- 優先度の高いエンドポイントやカスタムWebhook設定は有料
	    
	- 一部チェーンのサポートは限定的（Solana周辺はやや制限あり）

- ✅ 3. **Dune Analytics API**（https://dune.com/docs/api/）
	- APIの利用自体は有料で可視化までは無料
